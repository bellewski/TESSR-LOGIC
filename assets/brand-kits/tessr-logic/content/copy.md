## Value props (feature cards)
- **Fully Offline** — Runs entirely on your hardware. No CDN, no telemetry, no external calls. Built for air-gapped and classified environments.
- **Multi-Agent Pipeline** — Architect → Coder → UI Designer → Hardener → Fixer → Validator → Builder → Runtime QA → Design Critic. Each agent owns one job.
- **Self-Correcting** — Runtime QA executes every page; failures route a surgical, additive fix back to the responsible agent — it edits, never starts over.
- **Hardened by Default** — Security review and validation gates are part of the pipeline, not an afterthought.
- **Air-Gapped Ready** — No data leaves your perimeter. Purpose-built for DoD, government, and regulated enterprise.
- **Learns Your Standards** — An offline memory layer accumulates lessons from past builds, so output improves over time — privately.

## Pipeline agent blurbs
- **Architect** — Turns your requirement into a precise spec, contract, and file plan.
- **Coder** — Generates complete, working source code from the plan.
- **UI Designer** — Authors the full professional stylesheet and visual system.
- **Hardener** — Reviews code for security vulnerabilities.
- **Fixer** — Applies security and quality fixes.
- **Validator** — Confirms the build matches the spec.
- **Builder** — Installs, builds, and produces artifacts.
- **Runtime QA** — Executes pages headlessly to catch real runtime errors.
- **Design Critic** — Judges professional visual quality and routes polish fixes.
